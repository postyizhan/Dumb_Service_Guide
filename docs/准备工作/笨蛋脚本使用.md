# 笨蛋脚本使用

笨蛋脚本是由本 wiki 的作者之一 lilingfeng 编写的，专门为笨蛋设计的，你只需要双击启动，回答几个问题就能帮你自动完成。

## 功能

目前，我们已经编写了以下的笨蛋脚本

* 自动优化服务端配置文件
* 自动配置防矿透
* 自动配置防种子破解
* 自动生成启动脚本
* 自动安装常用插件
* 自动安装 Python
* 自动安装 Java
* 一键上传日志
* 自动配置反作弊
* 自动简单配置
* 自动选择服务端核心并下载
* 还有更多（在编写中）

## 使用准备


除生成启动脚本，其他脚本都必须启动一次服务端后才能使用，生成启动脚本需要提前安装服务器核心。

:::danger

所有笨蛋脚本都需要在服务器根目录执行!!

:::
